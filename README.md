# Responsive Web Architecture Project

## 📌 Project Title
Responsive Food Delivery Website

## 📖 Description
This project demonstrates responsive web architecture using a mobile-first approach. 
The website layout automatically adjusts for mobile, tablet, and desktop screens.

## 🚀 Features
- Mobile-first design
- Responsive layout using CSS Grid
- Media queries for different screen sizes
- Hamburger menu for mobile navigation
- Clean and simple UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📱 Responsive Design
- Mobile: Single column layout
- Tablet: Two-column layout
- Desktop: Three-column layout

## 📂 Project Structure

responsive-project/
│── index.html
│── style.css
│── script.js
│── README.md

## ▶️ How to Run Project
1. Download or clone this repository
2. Open index.html in any browser
